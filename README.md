#Sopa de Letras

## Descripción

Aplicación que genera una sopa de letras y muestra las coincidencias encontradas en ella

## Frameworks utilizados

	-Laraver 5.5
	-Bootstrap 4.0 Beta
	-JQuery 3.2.1

## Instalación

1. Extraer la carpeta *Sopa* en el directorio de su preferencia

2. Ejecutar los siguientes comandos en el directorio:
		- php artisan key:generate
		- php artisan serve

3. Ir a la siguiente [URL](localhost:8000).
